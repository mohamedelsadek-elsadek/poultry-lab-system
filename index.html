Backup saves all submissions, results and custom lists to a JSON file. Import restores from a backup.

Cloud sync (Firebase - optional)

Optional. Syncs submissions and lists across your devices/branches via your own Firebase Realtime Database. Data stays offline-first; sync is additive and merges by newest change.

Security note: an open database is readable by anyone with the URL. Use Firebase auth rules for real deployments. This runs entirely from your browser - no third party sees your data.

Bulk import farms (Excel / CSV)

Download the template, fill one row per submission, then upload it. Each row becomes a new submission. Columns: Sample Code, Company, Farm, Bird Type, Age (days), Region, Lab, Sample, Contact, Notes.

Lab column accepts the short code (VB, GD, AN, XO) or full name. Unknown labs default to the first lab.
